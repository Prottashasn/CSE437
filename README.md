# CSE437

📊 American Sign Language Alphabet Recognition Using PCA, MLP, CNN, and KNN: A Comparative Study.

This project was developed as part of the CSE 437: Data Science – Coding with Real-World Data course. It focuses on recognizing American Sign Language (ASL) alphabets from image data using machine learning and deep learning approaches.

The dataset consists of 14,500 labeled images of ASL alphabets, which were preprocessed and used to train multiple models for classification.

We implemented and compared the performance of:

1.Multilayer Perceptron (MLP)
2.Principal Component Analysis (PCA) + MLP
3.Convolutional Neural Network (CNN)
4.PCA + K-Nearest Neighbors (KNN)

📈 Key Results
1.Best Accuracy: PCA (75 components) + MLP → 89.24%
2.Raw MLP Accuracy: 87.86%
3.CNN Accuracy: 83.22%
4.PCA + KNN Accuracy: 81.89%

PCA significantly improved model efficiency while maintaining high accuracy, whereas CNN required more training time with comparatively lower performance in this setup.

💡 Project Goal

The goal of this project is to evaluate different models and identify the most effective approach for ASL alphabet recognition, contributing toward real-world assistive communication systems for the deaf and mute community.
